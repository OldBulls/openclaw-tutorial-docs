# OpenClaw 实战教程（公开版）

> 基于 20+ 天真实生产环境经历
> 覆盖：从零安装 → 多 agent 协作 → 记忆系统 → 飞书集成 → 运维 → 排障
> 适用版本：OpenClaw 2026.4.14+
> 最后更新：2026-04-18

OpenClaw 是一个**多 agent 协作运行时**，把 Claude / GPT 等模型组成一个 7×24 小时跑在你机器上的「家族」。本仓库是**作者本人**真实搭建过程的踩坑记录与方法论沉淀。

## 谁适合读

- 已装 OpenClaw、想搞清楚为什么 agent 没按预期协作
- 准备装但担心踩坑（运维 / 升级 / 飞书集成都很容易出错）
- 已经在用、想知道作者是怎么配多 agent 拓扑 + 记忆策略的

**不适合**：完全没用过 LLM 工具的纯小白（建议先看 OpenClaw 官方 quick start）。

## 进阶：付费内容

公开教程涵盖**方法论与原理**。如果你想跳过架构设计阶段、**直接拿到一套可用的多 agent 框架骨架**：

- `templates/`：去个性化的 SOUL / IDENTITY / MEMORY / AGENTS / USER 模板
- `agents/`：5+ 角色 agent 协作配置样例（main / planner / reflector / 业务 agent）
- `scripts/`：15 个生产级运维脚本模板（gateway 健康 / 日志轮转 / hotfix / 备份矩阵 / 周维护编排）
- `guided-install/`：问答式安装脚本，按你选择生成骨架（detect / prompt / apply / install / rollback 全套）
